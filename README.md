# meradOS Skills

Agent Skills fuer Claude Code nach dem Elon 5-Schritt-Algorithmus: Hinterfragen, Loeschen, Vereinfachen, Beschleunigen, Automatisieren.

## Skills

| Skill | Beschreibung |
|-------|-------------|
| **merados-desktop** | macOS nach Elon-Prinzip tunen - System, Terminal, Security, Keyboard, SSH, iTerm2, Docker, Git, Nerd Fonts, Raycast, Claude Code Settings |
| **merados-slack-advisor** | Projekt analysieren und Slack-Integrationen + Automatisierungen vorschlagen. Elon-Prinzip: nur was echten Wert liefert |
| **elon-algo** | Elon Musks 5-Schritt-Algorithmus auf jede Aufgabe anwenden |

## Installation

Einzeln:
```bash
npx skills add maf4711/merados-skills@merados-desktop
npx skills add maf4711/merados-skills@merados-slack-advisor
npx skills add maf4711/merados-skills@elon-algo
```

Alle:
```bash
npx skills add maf4711/merados-skills
```

## Philosophie

- Minimaler Code, maximale Wirkung
- Keine Abstraktion ohne Grund
- Loeschen vor Refactoring
- Reihenfolge ist kritisch: 1 Hinterfragen → 2 Loeschen → 3 Vereinfachen → 4 Beschleunigen → 5 Automatisieren
